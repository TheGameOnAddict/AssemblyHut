NEW
- Part Builder and Assembly Builder workspaces
- Insert saved parts into an assembly
- Insert saved assemblies into larger assemblies
- Arrange instances with move, rotate and scale
- Multi-select assembly items
- Save reusable assemblies in IndexedDB
- Reopen and update saved assemblies
- Duplicate and delete placed instances
- Circular assembly-reference protection
- Hierarchy-aware object selection
- Existing v0.9 definitions remain available

WORKFLOW
1. Create and save parts in Part Builder.
2. Switch to Assembly Builder.
3. Use Insert in the library for parts or existing assemblies.
4. Position the inserted items.
5. Optionally use Select Items to choose only some items.
6. Press Save Assembly.
7. Reopen an assembly with Edit.

Exploded-view setup arrives in v0.11.
